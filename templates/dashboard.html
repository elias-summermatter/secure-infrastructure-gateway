Before you download

This file unlocks access to protected infrastructure. It contains a private cryptographic key that identifies you personally.
  • Anyone with this file can impersonate you on the gateway.
  • Treat it like a password: never share it, email it, paste it into chat, or commit it to git.
  • Store it only on devices you control — not shared drives or cloud sync folders.
  • If the file ever leaks, come straight back here and click Rotate — the old one is invalidated instantly.
